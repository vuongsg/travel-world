# Introduction
A place where you can travel online around the world ^_^

## Installing
```
npm install
```
or
```
yarn install
```

## Running
```
npm start
```
or
```
yarn start
```

## Technologies
- ReactJS
- Redux
- TypeScript

## Browsers support
- Edge 16+
- Firefox 52+
- Chrome 57+
- Safari 10.1+
- Opera 44+
- Safari on IOS 10.3+
- Android Browser 90+
- Opera Mobile 62+
- Chrome for Android 90+
- Firefox for Android 87+
